How to Contribute

Follow @kcelebi Star WatchFork

For this project, I would like to discrimate between contributing small versus large chunks of code.

For small changes, such as typos and minor bugs, feel free to open pull request on GitHub. I'm quite active and will be able to attend to whatever changes you suggest quickly.

For large changes, which could involve creating/removing files, adding functions, or restructuring existing code, send me an email at kayacelebi17@gmail.com.
